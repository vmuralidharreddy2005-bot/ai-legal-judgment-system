# AI Legal Judgment Prediction System

An AI-powered legal document analysis platform using Natural Language Processing, Machine Learning, and Transformer Models.

## Features

- PDF Legal Document Upload
- NLP Text Preprocessing
- Legal Entity Extraction
- AI-Based Fraud Prediction
- Transformer-Based Summarization
- Semantic Similarity Search
- AI Legal Chatbot
- Analytics Dashboard

## Technologies Used

- Python
- Streamlit
- spaCy
- Scikit-learn
- Transformers
- SentenceTransformers
- Plotly

## How To Run

```bash
streamlit run app.py
